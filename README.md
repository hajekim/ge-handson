## 🏢 실습 시나리오
자동차/모빌리티, 로보틱스, 교육기관, 게임 등 다양한 산업군의 실무 시나리오를 활용합니다.

- **[자동차/모빌리티]** EU 전기차 정책 변화를 실시간 검색하여 SWOT 분석 매트릭스 도출
- **[로보틱스/제조]** CES 신기술 동향을 분석하여 경쟁사 대응 핵심 마케팅 메시지 수립
- **[마케팅/광고]** 브랜드 캠페인 포스터 이미지 생성 및 Veo 시네마틱 루프 영상 제작
- **[소비재/유통]** 글로벌 커피 원두 매출 데이터를 다각도로 분석하고 시계열 트렌드 예측
- **[IT/클라우드]** 수십 페이지 분량의 신규 기능 발표 장표를 요약하고 실무 항목으로 분류
- **[교육기관]** 대학 플립러닝 수업 설계 및 교육부 국책사업 기획안 작성
- **[게임/엔터테인먼트]** MMORPG 여름 한정 이벤트 섬 콘텐츠 및 퀘스트 기획
- **[범용 기업]** 뉴스 링크 하나로 SNS 홍보 문구를 자동 생산하는 노코드 에이전트 빌딩
