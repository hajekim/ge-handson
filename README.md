## 🏢 글로벌 모빌리티 비즈니스 대표 실습 시나리오
본 마스터 과정의 실습은 **현대자동차(Hyundai Motor Group)**, **제네시스(Genesis)**, **보스톤 다이내믹스(Boston Dynamics)**의 글로벌 마케팅, 생산 전략, 제품 혁신 등의 생생한 비즈니스 유스케이스를 관통하는 유기적인 여정으로 구성되어 있습니다.

- **마케팅 트렌드 분석**: 북미/유럽 프리미엄 세단 시장 경쟁 우위 분석 및 안전한 동료 공유
- **공식 보도자료 분석**: 실시간 웹 검색 그라운딩 기반 EU 전기차 정책 동향 및 보스톤 다이내믹스 대응 SWOT 수립
- **미디어 생성**: Imagen 3를 사용한 제네시스 마케팅 포스터 생성 및 Veo 모델을 활용한 시네마틱 루프 제작
- **매출 다차원 데이터 분석**: 커피 원두 글로벌 매출 데이터를 다각도로 추적 및 시계열 트렌드 예측
- **신규 기능 PPTX 완벽 요약**: 수십 페이지의 BigQuery 신 GA 기능 발표 장표 요약 분석 및 실무 분류
- **에이전트 제작**: 뉴스 링크 하나로 SNS 마케팅 홍보용 문구를 자동 양산하는 자율 에이전트 빌딩
